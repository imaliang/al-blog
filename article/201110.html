Markdown基本语法

本文最后更新于：2021年1月21日 下午

Markdown是一种轻量级的「标记语言」，适合追求简而至上的开发者。

1. 标题

# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题

2. 字体

**加粗**
*倾斜*
***斜体加粗***
~~加删除线~~

加粗
倾斜
斜体加粗
加删除线

3. 引用

> 一级引用
>> 二级引用
>>> 三级引用

一级引用

二级引用

三级引用

4. 表格

表头|表头|表头
:---|:--:|---:
居左|居中|居右
内容|内容|内容
姓名表头表头
路飞橡胶果实海贼王

5. 空白行

<br/>

6. 图片

![图片alt](图片地址 "图片title")

图片alt就是显示在图片下面的文字，相当于对图片内容的解释。
图片title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加

7. 超链接

[超链接名](超链接地址 "超链接title")

阿椋博客

8. 列表

无序列表： - + * 任何一种都可以
如果需要下一级 敲三个空格即可
加 [ ] 可实现复选框效果，x 代表选中，留空为不选中

- 列表内容
   - 列表内容
+ [ ] 列表内容
* [x] 列表内容
  • 列表内容
    • 列表内容
  • 列表内容
  • 列表内容

有序列表： 数字加点（序号跟内容之间要有空格）

1. 列表内容
2. 列表内容
3. 列表内容
  1. 列表内容
  2. 列表内容
  3. 列表内容

9. 代码

单行代码：代码之间分别用一个反引号包起来
System.out.println("阿椋博客");

代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行

System.out.println("阿椋博客");

10. 分割线

三个或者三个以上的 - 或者 * 都可以。

---
----
***
*****

11. 特殊字符

&emsp; 				// 全角的空格（一个中文的宽度）
&ensp;				// 半角的空格（半个中文的宽度）
&thinsp;    		// 窄空格（1/6个中文的宽度）
&nbsp;				// 不断行的空格 该空格占据宽度受字体影响明显而强烈
&zwnj; 				// 抑制本来会发生的连字效果
&zwj;				// 使两个本不会发生连字的字符产生连字效果
